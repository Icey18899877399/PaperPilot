# 接口清单

| 方法 | 地址 | 用途 |
|---|---|---|
| GET | /api/health | 服务健康检查 |
| POST | /api/papers | 上传并解析PDF |
| POST | /api/papers?background=true | 上传PDF后立即返回，由后台执行MinerU解析 |
| GET | /api/papers | 获取论文列表 |
| GET | /api/papers/{paper_id} | 获取论文状态 |
| POST | /api/papers/{paper_id}/retry | 重新解析失败论文并重新生成导读 |
| DELETE | /api/papers/{paper_id} | 删除论文及PDF、解析资源、索引、导读和思维导图 |
| GET | /api/papers/{paper_id}/contents | 按类型或页码获取MinerU结构化内容 |
| GET | /api/papers/{paper_id}/guide | 获取自动生成并持久化的论文导读 |
| POST | /api/papers/{paper_id}/guide | 读取缓存或调用论文理解Agent重新生成导读 |
| POST | /api/papers/{paper_id}/mind-map | 调用论文理解Agent生成思维导图 |
| GET | /api/papers/{paper_id}/mind-map | 获取已持久化的思维导图 |
| POST | /api/papers/{paper_id}/translate | 调用翻译Agent |
| POST | /api/chat | 调用聊天Agent进行RAG问答 |
| GET | /api/videos | 获取 B 站视频资源目录 |
| POST | /api/videos | 登记 B 站视频链接或搜索入口 |
| PUT | /api/videos/{video_id} | 编辑视频标题、简介、B站链接、来源和匹配元数据 |
| DELETE | /api/videos/{video_id} | 删除视频记录 |
| GET | /api/agents/logs | 查询多Agent调用日志 |
| GET | /api/models/status | 查询模型配置状态，不返回密钥 |
| POST | /api/models/test | 发起最小模型连接测试 |
| GET | /media/papers/{filename} | 访问上传的PDF |
| GET | /media/videos/{filename} | 访问默认视频封面等静态资源 |

具体请求和响应结构以运行后的FastAPI OpenAPI页面为准。

后台解析时论文记录中的`status`为`parsing`，`progress`提供0—100的估算进度，`stage`说明当前阶段。前端每2秒查询`GET /api/papers/{paper_id}`，完成后状态变为`ready`；失败时状态为`failed`并返回`error`。

上传接口默认限制PDF不超过50MB，并校验PDF文件头。系统使用SHA-256内容哈希检测重复论文；重复上传返回HTTP 409。解析完成后自动生成并保存智能导读，失败论文可通过重试接口复用原始PDF重新执行解析。
